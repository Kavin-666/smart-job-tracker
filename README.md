# Smart Job Tracker 🚀

A modern and intuitive **job application tracking system** built with React to help you stay organized, focused, and in control of your job search.

---

## 📌 Overview

Smart Job Tracker is designed to simplify the entire job search process.
Instead of losing track of applications across emails and spreadsheets, this app provides a **centralized dashboard** to manage everything—from applications to interviews and offers.

It helps you:

* Track progress visually 📊
* Stay organized 📂
* Make data-driven decisions 📈

---

## ✨ Features

### 📊 Dashboard

* Real-time overview of your job search
* Total applications, interviews, and offers
* Salary insights and rejection rate

### 📋 Application Management

* Add, edit, and delete job applications
* Track company, role, status, and notes
* Clean and structured UI

### 🔍 Search & Filters

* Quickly find applications
* Filter by job status (Applied, Interview, Offer, Rejected)

### 📈 Analytics

* Visual insights into your job search
* Monthly applications tracking
* Pipeline breakdown charts

### ⚡ Quick Actions

* Add new applications instantly
* Navigate between features easily

### 🌙 Dark Mode

* Toggle between light and dark themes for better UX

---

## 🛠️ Tech Stack

* **Frontend:** React + Vite
* **State Management:** Context API + Custom Hooks
* **Styling:** CSS
* **Data Handling:** Local Storage

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Kavin-666/smart-job-tracker.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the app

```bash
npm run dev
```
